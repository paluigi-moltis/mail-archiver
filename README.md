# Mail Archive Daemon

Self-hosted, Dockerized, single-user email archiving solution with hybrid search.

![Rust](https://img.shields.io/badge/Rust-2021-orange.svg)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-16+-336791.svg)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## Overview

Connect to multiple IMAP accounts, download emails and attachments, store permanently, and search with hybrid keyword + semantic search.
